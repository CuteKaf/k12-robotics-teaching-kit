# 课程总纲（Course Outline）

## 课程名称
K-12 机器人系统实践课：从 0 到 1 搭建能看、能判、能动的机器人系统

## 课程定位
本课程是一套以线下授课为主、视频辅助为辅的项目制机器人课程。围绕真实工程项目展开，通过 3D 打印、机械臂装配、Ubuntu 环境、机器视觉、坐标变换与流程闭环，引导学生完成从软件到硬件的完整实践。

## 课程目标

### 知识目标
- 理解机器人系统的输入、决策、执行闭环
- 理解 Ubuntu、Python、Conda、YOLO、RealSense、手眼标定等核心概念
- 理解相机坐标系、机械臂坐标系与安全控制逻辑

### 能力目标
- 能完成环境搭建与基础自检
- 能跑通视觉检测与深度坐标输出
- 能完成标定、悬停、假抓取等关键验证步骤
- 能阅读文档、定位问题、进行基础排错

### 素养目标
- 培养工程拆解能力与小步验证意识
- 培养团队协作与文档阅读能力
- 建立真实项目与竞赛/论文转化的认知

## 课程模块

### M1 机器人与编程基础
- Ubuntu / Linux 基础
- Python / Conda
- 项目目录理解
- 环境初始化与自检

### M2 机械系统与 DIY 装配
- 3D 打印基础
- SO-ARM100 结构理解
- 电机、控制板、末端执行器装配
- 安全与连接规范

### M3 机器视觉：机器人之眼
- RealSense 取流
- YOLO 2D 检测
- RGB-D 与目标 XYZ 输出

### M4 运动学：坐标系与位姿
- 基坐标系、工具坐标系、相机坐标系
- task pose 与 drop pose
- 位姿记录与工作空间约束

### M5 系统集成与流程闭环
- 手眼标定
- 视觉悬停
- fake grasp
- 总控骨架理解

### M6 进阶与创新挑战
- Web 控制台
- 分类扩展
- 竞赛项目包装
- 论文成果转化

## 教学方式
- **线下主讲**：装配、调试、理解、排错
- **视频预习**：基础概念、标准操作
- **视频复习**：总结、常见错误、漏课补看
- **任务驱动**：每节课必须有成功标准

## 课程节奏建议
- 12 周版：每周 1 次
- 24 节版：每周 2 次或每次拆成上下半节

## 适合对象
- 初中高年级 / 高中生
- 科创比赛、机器人竞赛准备学生
- 对 AI + 机器人系统实践有兴趣的学生
