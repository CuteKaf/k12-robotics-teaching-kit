# 课程简介（给家长/招生）

这是一套面向 K-12 / 高中阶段的机器人系统实践课程。

学生将在课程中完成：
- 3D 打印机械结构件
- 自主装配机械臂与末端执行器
- 搭建 Ubuntu / Python / AI 开发环境
- 使用 RealSense 相机获取 RGB-D 数据
- 使用 YOLO 进行目标识别
- 理解坐标系、标定与机械臂控制
- 完成从视觉识别到动作执行的闭环验证

## 课程特色
- 项目驱动，不是纸上谈兵
- 小步验证，降低挫败感
- 强调工程实践、调试和排错
- 支持竞赛、项目展示与论文成果转化

## 教学形式
- 线下授课为主
- 视频与任务单辅助学习
- 每节课有可见成果
