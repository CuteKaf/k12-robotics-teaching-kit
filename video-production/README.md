# Video Production

这个目录专门讨论一个问题：

**在线下机器人课程里，视频应该怎么做。**

这里不把视频当成主课本体，而是当成：
- 课前预习材料
- 课堂标准操作回放
- 课后复盘材料
- 漏课补看材料
- 助教统一口径工具

## 你最需要先理解的事
在这类课程里，视频的价值不是“替老师上课”，而是：

1. 降低课堂里重复讲基础内容的时间
2. 让学生课后能继续跟上
3. 让助教有统一说明口径
4. 让课程更容易复制和扩班
5. 让家长能看见学习过程与成果

## 本目录包含
- 为什么视频是辅助而不是主体
- 哪些内容该录，哪些不该录
- 视频的三种类型如何分工
- 最省力的录制方式
- 最适合你的剪辑思路
- 每条视频的统一模板

## 使用顺序建议
1. 先读 `why-video-is-supporting-not-main.md`
2. 再读 `what-to-film-and-what-not-to-film.md`
3. 再读 `video-types-for-offline-robotics-course.md`
4. 然后看 `recording-workflow.md`
5. 最后用 `template-for-each-video.md` 开始实际录制
