# 录屏 + 配音 + 字幕 + 轻剪辑 SOP

## 一、录制前
1. 先确定这条视频只讲一个问题
2. 写一个极短提纲
3. 准备好成功画面
4. 统一终端字体、缩放、主题

## 二、录屏
1. 先安静录操作
2. 操作尽量一次走顺
3. 遇到明显失误可以重来
4. 成功结果停留 2~3 秒

## 三、配音
1. 按提纲补讲解
2. 不要边录边即兴讲太多
3. 重点讲：为什么、怎么做、什么算成功

## 四、字幕
1. 先自动转字幕
2. 再统一术语
3. 修正命令、路径、包名、设备名
4. 保证断句清晰

## 五、轻剪辑
1. 删掉明显失误
2. 删掉无意义等待
3. 保留关键过程
4. 加标题、字幕、高亮和错误提示

## 六、导出
1. 命名统一
2. 放进对应目录
3. 标记用途：预习 / 课堂辅助 / 复盘

## 最重要的原则
不要把每条视频都当作品去精雕细琢。
先把它做成稳定、可复用、可持续录制的教学资产。
